**别名：`blitz d`**

启动 Blitz 开发环境服务器。

#### 选项

| 选项                     | 简写 | 描述                           | 默认          |
| ------------------------ | ---- | ------------------------------ | ------------- |
| `--hostname`             | `-H` | 设置服务器要使用的 hostname    | `"localhost"` |
| `--port`                 | `-p` | 设置你想让服务器监听的端口     | `3000`        |
| `--no-incremental-build` |      | 禁用增量构建并从一个新缓存开始 | `false`       |
| `--inspect`              |      | 启用 Node.js inspector         | `false`       |

#### 示例

```bash
blitz dev
blitz dev -p 4000
```
