在当前目录中生成一个新的 Blitz 项目。该项目将包括登录页面的基本脚手架、数
据库以及使 Blitz 的魔法发挥作用的所有其他配置。

```bash
blitz new [name]
```

| 参数   | 必须 | 描述                                   |
| ------ | ---- | -------------------------------------- |
| `name` | 是   | 项目名称。将被用于初始化的项目脚手架中 |

#### Options

| 参数        | 简写 | 描述                                                                                      | 默认    |
| ----------- | ---- | ----------------------------------------------------------------------------------------- | ------- |
| `--npm`     |      | 使用 `npm` 作为项目默认的包管理器。如果省略，在已安装 Yarn 的前提下将使用 `yarn` 来安装。 | `false` |
| `--dry-run` | `-d` | 显示将生成哪些文件但不将文件写入磁盘。                                                    | `false` |
| `--no-git`  |      | 跳过生成 Git 仓库。                                                                       | `false` |

#### 示例

```bash
blitz new task-manager
```

```bash
blitz new task-manager --npm
```
