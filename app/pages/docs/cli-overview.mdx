Blitz CLI 是你与应用从数据库管理到代码生成等进行交互的一体化访问入口。它将
通过专制的决策来引导你走向最佳实践，同时又足够灵活，让你可以随心所欲地构建
项目。

### 安装 {#installation}

在设置应用时，你可能已经与 Blitz CLI 进行了交互。我们建议全局安装 Blitz 以
确保你可以快速按需运行 Blitz 命令：

```bash

# npm
npm i -g blitz

# Yarn
yarn global add blitz
```

### 可用命令 {#available-commands}

在接下来的章节中，你可以了解所有 CLI 命令。

此外，你可以运行 `blitz help` 或 `blitz [COMMAND] --help` 以直接在终端获取
帮助。
